Network Workflow Builder

Web-based drag-and-drop workflow builder for network device configuration

Key Features
  • Drag & Drop Interface: Visual workflow builder
  • Template System: Reusable configuration templates
  • Dynamic Task Insertion: Insert tasks anywhere in workflow
  • Real-time Updates: WebSocket-based live updates
  • Netmiko Integration: Execute CLI commands on devices
  • Multi-Device Support: Cisco, Arista, Juniper, and more
  • Variable Management: Dynamic variable substitution
  • REST API: Complete API for automation

REST API Endpoints
Method Endpoint Description
GET /api/workflows List all workflows
POST /api/workflows Create new workflow
PUT /api/workflows/{id} Update workflow
POST /api/workflows/{id}/execute Execute workflow
POST /api/workflows/{id}/tasks/{taskId}/insert-after Insert task after another task
GET /api/templates List all templates
